Senior Front-End Engineer

Full Time ‒ San Mateo, CA

The Tachyus platform provides rich data visualization and analysis capabilities in the browser using D3 and Kendo UI. The Tachyus platform delivers HTML5 experiences rivaling native applications in performance and elegance. We build our front-end applications in JavaScript and are in the process of transitioning parts to TypeScript.

The ideal candidate has deep experience building web applications, particularly using the single-page application (SPA) paradigm. The candidate has experience delivering enterprise applications with challenging technical requirements and is able to communicate and promote best practices within a front-end development team. Additionally, a very well qualified applicant will have experienced with TypeScript and feel comfortable reading our back-end F# code to understand the mechanics of the APIs consumed by the front-end.

Responsibilities

  • Participate in the architecture design of Tachyus front-end web applications
  • Write, test, and maintain JavaScript and TypeScript code
  • Work closely with our visual design team to construct responsive user experiences
  • Collaborate with Full Stack engineers to design web APIs for web applications
  • On-board, train, and mentor junior front-end developers
  • Understand our users and build features to meet their requirements

Requirements

  • 5+ years experience in client development. Web experience preferred
  • Deep knowledge of HTML, CSS, and JavaScript
  • Some experience with TypeScript or another compile-to-JavaScript compiler, such as Closure, Traceur, 6to5, or Regenerator
  • Experience unit testing, integration testing, and cross-browser testing
  • Experience building a complex, single-page web application
  • Desire to work with a rapidly growing, values-driven start-up
  • Open-minded technology and platform perspective: we’re one of the top F# shops in the world, and that means this is a partially Microsoft environment

Extras

  • Experience with D3, Kendo UI or other data visualization libraries
  • Deep experience with TypeScript or other compile-to-JavaScript languages
  • Experience developing for and supporting mobile applications
  • Full-stack development experience; demonstrated interest in functional programming
  • Demonstrated technical leadership skills
